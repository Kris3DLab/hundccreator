# 🤖 Magyar DC Bot Creator v6.0
A legegyszerűbb, blokk-alapú Discord bot készítő szoftver magyar nyelven.

## ✨ Funkciók
* **Drag & Drop** felület: Húzd be a blokkokat, és kész a botod!
* **Auto-Update**: Mindig a legfrissebb verziót használod.
* **Moderáció**: Kick, Ban, Mute, Purge blokkok beépítve.
* **EXE formátum**: Nem kell Python a futtatáshoz.

## 🚀 Telepítés
1. Menj a [Releases](https://github.com/Kris3DLab/hundccreator/releases) menüpontba.
2. Töltsd le a legfrissebb `main.exe` fájlt.
3. Indítsd el, másold be a Token-ed, és tervezd meg a botodat!

## ⚠️ Fontos
A bot futtatásához a [Discord Developer Portalon](https://discord.com/developers/applications) be kell kapcsolnod mind a három **Privileged Gateway Intent**-et!
